

###  chrome 书签同步文件夹


``` 使用说明
使用他人开源项目 
https://github.com/wujunxi/bookmark/tree/e1d3ee8df6c084c457a47b46e6f5da3a608fc9dd

git cloen 将项目下载 获得其中src.crx 把后缀改为rar解压为一个文件使用chrome加载解压文件将其装上插件


在github中获取token  生成的 token 需要勾选 repo 权限
token获取地址 https://github.com/settings/tokens/new

创建一个项目仓库,不设置默认自动创建为my_bookmarks/data.json

返回chrome 选择插件图标 填写github账号,token,以及创建的参考路径

选择记住 remember me 账号

选择 upload 同步

```
